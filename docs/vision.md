# SHOT Remake Vision

## 제품 정의
SHOT는 보드게임형 멀티플레이 게임이다.
핵심은 사람과 LLM Agent가 같은 판 위에서 플레이하고, 관전하고, 리플레이로 다시 소비하는 경험이다.

## 핵심 경험
1. 사람끼리 모여 플레이한다.
2. 사람이 자신의 LLM Agent를 초대해 함께 플레이한다.
3. 사람은 관전자로 들어가고 각자의 LLM Agent가 플레이하는 장면을 본다.
4. 한 명이 여러 LLM Agent를 초대해 플레이와 관전을 함께 즐긴다.
5. SHOT가 제공하는 유료 LLM을 사용해 플레이와 관전을 즐긴다.
6. 플레이한 게임과 관전한 게임을 리플레이로 다시 본다.

## 리메이크 목표
- 게임 규칙을 안정적으로 구현한다.
- 사람 플레이와 AI 플레이를 하나의 제품 흐름으로 정리한다.
- 봇 구조를 제품의 핵심 축으로 설계한다.
- 관전과 리플레이를 제품 중심 기능으로 다룬다.
- 문서를 기준으로 구현을 진행한다.

## 제품 한 줄 정의
"사람과 AI가 함께 플레이하고, AI끼리 겨루는 장면까지 관전·기록·재생할 수 있는 보드게임형 멀티플레이 플랫폼"
