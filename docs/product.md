# Product Definition

## 사용자 유형
- 방을 만들고 친구들과 플레이하는 일반 유저
- 자신의 AI 동료 봇과 함께 플레이하고 싶은 유저
- 운영/밸런스/안정성을 관리하는 관리자

## 핵심 가치
- 간단한 진입
- 짧고 명확한 게임 흐름
- 관전자/참가자 상태가 분명한 실시간 경험
- "내 봇이 나를 따라온다"는 정체성 있는 경험

## 이번 리메이크 범위
### 포함
- 회원가입/로그인
- 방 생성/참가
- 게임 시작/종료
- 실시간 상태 동기화
- follow-owner 봇 참가
- 봇 상태 표시
- 운영 가능한 배포 구조

### 제외
- autonomous bot auto-join
- 과도한 경제 시스템
- 다중 게임 모드 동시 개발
- 복잡한 확장보다 우선되는 부가 기능

## 성공 조건
- 사람이 방을 만들고 게임을 문제 없이 끝낼 수 있다.
- 주인이 방에 들어가면 연결된 봇도 같은 방에 따라 들어올 수 있다.
- 게임 상태 동기화가 안정적이다.
- 개발자가 문서만 읽고 구조를 이해할 수 있다.
