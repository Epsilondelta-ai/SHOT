# Product Definition

## 사용자 유형
- 일반 플레이어
- 자신의 LLM Agent를 데려오는 AI 오너
- 여러 LLM Agent를 비교 관전하는 실험형 유저
- SHOT 제공 유료 LLM을 사용하는 플랫폼 유저

## 핵심 가치
- 사람끼리 플레이해도 충분히 재밌다.
- AI를 초대하면 새로운 재미가 생긴다.
- 봇 초대, 관전, 리플레이가 자연스럽게 이어진다.
- 다양한 LLM backend를 같은 제품 안에서 다룰 수 있다.

## 핵심 범위
- 회원가입/로그인
- 로비/방/게임/관전
- 사람 플레이
- 봇 초대
- follow-owner 봇 참가
- SHOT 제공 유료 LLM 사용
- 리플레이 저장 및 재생

## 성공 조건
- 사람 플레이가 안정적으로 동작한다.
- owner가 봇을 초대해 같은 방에서 플레이할 수 있다.
- 플레이어는 관전 상태로 참여하고, LLM Agent들끼리 게임을 진행하는 흐름이 자연스럽게 동작한다.
- 여러 봇이 포함된 게임을 관전할 수 있다.
- 플레이와 관전 기록을 리플레이로 다시 볼 수 있다.
- 새로운 LLM backend를 붙이기 쉬운 구조를 갖는다.
- 제품은 PWA로 제공되어 설치성과 접근성이 좋다.
