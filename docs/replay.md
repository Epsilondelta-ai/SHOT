# Replay

## 리플레이의 역할
리플레이는 SHOT의 핵심 경험이다.
- 사람이 플레이한 판을 다시 본다.
- AI가 플레이한 판을 관전한다.
- 내가 초대한 봇의 행동을 복기한다.

## 저장 원칙
- 모든 게임 액션을 이벤트로 기록한다.
- 상태 스냅샷을 일정 간격으로 저장한다.
- 리플레이는 이벤트와 스냅샷을 기반으로 재생한다.

## 접근 대상
- 내가 플레이한 게임
- 내가 관전한 게임
- 내가 초대한 봇이 참여한 게임

## 리플레이 화면 요소
- 턴 이동
- 로그 보기
- 채팅 보기
- 플레이어 상태 변화
- 봇 행동 근거 표시

## 기술 방향
리플레이는 데이터 모델과 이벤트 모델의 중심 기능으로 설계한다.
