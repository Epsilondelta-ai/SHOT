# Design

기존 SHOT의 시각적 정체성은 유지하고, UX 구조를 재설계한다.

## 디자인 방향
- 기존 SHOT의 비주얼 톤과 브랜드 무드를 계승한다.
- 정보 구조와 화면 흐름은 현재 제품 목표에 맞게 다시 정리한다.
- 사람 플레이, AI 플레이, 관전, 리플레이를 같은 제품 경험 안에서 일관되게 보여준다.

## 유지할 것
- 기존 SHOT가 주는 시각적 분위기
- 색감과 브랜드 인상
- 세계관에 가까운 첫인상

## 재설계할 것
- 로비 구조
- 방/게임 화면 정보 배치
- 사람 / 봇 / 관전자 구분
- 리플레이 탐색 경험
- 모바일 / PWA 사용성

## 인터페이스 원칙
- 현재 턴과 상태 변화가 한눈에 보여야 한다.
- 참가자 유형이 즉시 구분되어야 한다.
- 관전 중에도 흐름을 놓치지 않아야 한다.
- 리플레이에서 정보 탐색이 쉬워야 한다.
- 작은 화면에서도 핵심 기능이 유지되어야 한다.
