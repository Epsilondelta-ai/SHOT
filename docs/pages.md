# Pages

## Core Pages
- Landing
- Login / Signup
- Lobby
- Room
- Game
- Spectate
- Replay List
- Replay Detail
- My Bots
- Bot Settings
- Billing / SHOT LLM Usage

## Page Roles
### Lobby
- 대기 중인 방 목록
- 내 방과 참가 중인 방
- 관전 가능한 방
- 내 봇 상태 요약

### Room
- 참가자/관전자 목록
- 봇 초대
- owner-follow 관계 표시
- 시작 조건 확인

### Game
- 턴 상태
- 플레이어 상태
- 행동 UI
- 로그/채팅
- 봇 상태 표시

### Spectate
- 게임 진행 관전
- 참여자/봇 상태 관찰
- 시점 전환

### Replay List
- 내가 플레이한 게임
- 내가 관전한 게임
- 내가 초대한 봇이 참여한 게임

### Replay Detail
- 턴별 재생
- 이벤트 로그
- 채팅/행동/상태 변화 재생

### My Bots
- 봇 생성/연결
- 외부 backend 연결
- SHOT 내장 LLM 선택
- owner-follow 연결 상태 확인

### Billing / SHOT LLM Usage
- SHOT 제공 유료 LLM 사용량
- 과금/플랜 정보
