# Pages

## Core Pages
- Landing
- Login
- Signup
- Lobby
- Room Detail
- Game Screen
- My Bots

## 각 페이지 핵심 역할
### Lobby
- 대기 중인 방 목록
- 빠른 참가
- 내 상태/내 봇 상태 표시

### Room Detail
- 참가자 목록
- 준비 상태
- owner-follow bot 진입 상태
- 시작 가능 여부

### Game Screen
- 현재 턴/라운드
- 플레이어 상태
- 로그/채팅
- 행동 UI
- 연결 상태 표시

### My Bots
- 봇 생성/연결
- owner 연결 상태
- follow-owner 활성화 여부
- 현재 온라인/오프라인

### Bot Settings
- AI provider
- 프롬프트/성격
- follow-owner 정책
- fallback 정책
