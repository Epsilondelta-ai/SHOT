# CoAI Compatibility Layer

This project was initialized by coai.

## Intent
- Preserve the familiar MoAI command vocabulary inside Codex.
- Treat `.coai/` as the public project contract.
- Use Codex as the execution engine.

## Suggested usage
- `coai project`
- `coai plan "Describe the task"`
- `coai run SPEC-EXAMPLE-001`
- `coai sync SPEC-EXAMPLE-001`
